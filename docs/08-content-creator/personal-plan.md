# 路径 08：技术自媒体 — 个人计划书

## 5年阶段目标

### Y1：起步 — 建立内容节奏
- 每周2篇技术文章(公众号+知乎)
- 每月1个B站视频
- 粉丝目标：5万
- **里程碑**：副业收入¥35万

### Y2：增长 — 多元变现
- 开设知识星球/付费专栏
- B站粉丝10万+
- **里程碑**：收入¥180万

### Y3：规模化 — 全职创作或IP公司
- 招1-2人团队(剪辑/运营)
- 出版第一本书
- **里程碑**：收入¥600万

## 关键行动

1. **前100天**：每天发布一条技术短内容(知乎想法/推特)
2. **前6月**：积累50篇深度技术文章
3. **Y1结束**：有3篇10万+阅读爆款

## B计划
- Y1粉丝<1万→内容方向不对,调整或放弃
- Y2收入<¥50万→配合路径02(大厂)或04(咨询)做
