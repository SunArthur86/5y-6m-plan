# 路径 02：大厂 AI 技术高管 — 个人计划书

## 5 年分阶段目标

### Y1：入局 — 拿到大厂P8 offer
- 系统复习算法+系统设计面试（3个月）
- 瞄准字节/阿里/小红书AI团队
- 入职后6个月站稳脚跟
- **里程碑**：拿到P8 offer，年薪150-250万

### Y2：证明 — 成为团队核心
- 主导一个公司级AI项目
- 带领3-5人子团队
- 发表1-2篇技术演讲（QCon/ArchSummit）
- **里程碑**：获得晋升P8+资格

### Y3：突破 — 冲击P9
- 管理完整团队（20-30人）
- 推动跨部门AI基础设施项目
- 在行业建立技术影响力
- **里程碑**：进入P9晋升通道

### Y4：跃迁 — 稳固P9
- 管理多个团队（50-100人）
- 定义团队技术战略
- 建立行业人脉网络
- **里程碑**：P9确认，年薪500万+

### Y5：登顶 — 向VP迈进
- 管理大部门（100-200人）
- 影响公司AI战略方向
- 考虑是否跳VP或转创业
- **里程碑**：年薪600万+

## 关键行动清单

**Y1 入职前（现在→入职）：**
1. 系统设计面试准备（每日1题×90天）
2. 刷LeetCode中等+困难200题
3. AI/LLM前沿论文每周2篇
4. 模拟面试10次以上

**Y1 入职后：**
1. 前30天：了解团队+技术栈+关键项目
2. 前90天：独立承担一个中等项目
3. 6个月：主导一个重要项目
4. 12个月：拿到solid rating
