# 路径 21：AI + 机器人 / 具身智能 — 商业计划书

## 一句话定位

做具身智能（Embodied AI）的软件大脑——为机器人提供AI决策/感知/操作能力。

## 市场分析

| 指标 | 数据 |
|------|------|
| 全球机器人市场 | $500亿(2025) |
| 具身智能融资(2024) | $30亿+ |
| 年增长率 | 30%+ |
| 代表公司估值 | Figure($26亿)/1X($5亿) |

## 为什么现在入场

- 2024-2025是具身智能元年
- 硬件成本下降（传感器/电机/芯片）
- 大模型给机器人"大脑"（VLA模型）
- 软件门槛降低，但AI软件仍是核心瓶颈

## 产品方向

### 方向A：机器人AI操作系统
- 通用机器人AI中间件
- 感知(VLM)+规划(LLM)+控制(RL)
- 类似：ROS的商业化AI版本
- B2B授权：¥50-500万/年

### 方向B：特定场景机器人方案
- 仓储物流机器人(分拣/搬运)
- 餐饮服务机器人(端菜/清洁)
- 全栈方案(软件+集成)
- 单项目：¥100-1000万

### 方向C：仿真训练平台
- 数字孪生+仿真训练环境
- 让机器人在虚拟世界学习
- 类似：NVIDIA Isaac Sim
- SaaS：¥5-20万/年

## 风险

| 风险 | 应对 |
|------|------|
| 硬件依赖强 | 纯做软件层，不碰硬件 |

| 行业太早期 | 先做特定场景落地 |
