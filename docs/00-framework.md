# 执行计划框架说明

## 时间维度

| 维度 | 范围 | 用途 |
|------|------|------|
| 5年战略 | Y1-Y5 | 大方向，每年1个里程碑 |
| 年度计划 | 12个月 | 4个季度OKR |
| 月度计划 | 4周 | 每周3-5个关键任务 |
| 周计划 | 7天 | 每日行动清单 |
| 日复盘 | 当天 | 完成度+明日计划 |

## 优先级标记

- 🔴 P0：必须完成，阻塞后续所有任务
- 🟡 P1：重要，本周必须推进
- 🟢 P2：可以做，有余力才做
- ⚪ P3：想法池，记录但不执行

## 通用周节奏

| 日 | 活动 | 时长 |
|----|------|------|
| 周一 | 本周计划+P0任务启动 | 2h计划+6h执行 |
| 周二-四 | 核心执行时间 | 每天8-10h |
| 周五 | 收尾+复盘+下周准备 | 6h |
| 周六 | 学习/副业/人脉拓展 | 6h |
| 周日 | 休息+复盘+阅读 | 4h |

## 通用月度复盘模板

```
[月份] 复盘
━━━━━━━━━━━━━━━━━━
✅ 完成的（列出来，量化）
❌ 未完成的（列出来，分析原因）
📊 关键指标（根据路径不同）
🎯 下月3个最重要目标
💡 本月最大的收获/教训
```
