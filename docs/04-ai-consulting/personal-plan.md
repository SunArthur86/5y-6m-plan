# 路径 04：AI 技术咨询 — 个人计划书

## 5 年分阶段目标

### Y1：起步 — 建立个人品牌
- 前6月：在大厂/工作中积累案例+内容输出
- 后6月：兼职接2-3个咨询项目
- **里程碑**：副业收入¥30万

### Y2：全职 — 独立咨询师
- 离职全职做咨询
- 签3-5个长期顾问客户
- **里程碑**：年收入¥490万

### Y3：规模化 — 建立工作室
- 招2-3名助理/初级顾问
- 扩大客户规模
- **里程碑**：年收入¥1100万

### Y4-Y5：转型 — 从时间换钱到杠杆收入
- 开发标准化AI咨询产品
- 课程/培训收入占比提升
- **里程碑**：年净收入600万+
