# 路径 04：AI 技术咨询 — 商业计划书

## 一句话定位

为企业提供AI技术架构咨询+落地实施服务，按项目或按天收费，3-5万/天。

## 市场分析

| 指标 | 数据 |
|------|------|
| 中国企业AI转型市场规模 | ¥5000亿/年 |
| AI咨询需求增长率 | 50%/年 |
| 顶级技术顾问日薪 | ¥3-10万 |
| 客单价（项目制） | ¥30-300万 |

**核心洞察**：90%企业想用AI但不知道怎么用，愿意花钱请专家。

| 服务 | 价格 | 交付物 | 周期 |

| 类型 | 示例 | 痛点 | 预算 |

## 获客渠道

1. **内容营销**（40%客户）：技术文章/演讲→客户主动联系
2. **人脉推荐**（30%客户）：前同事/客户转介绍
3. **合作伙伴**（20%客户）：SaaS厂商/集成商转包
4. **招投标**（10%客户）：政府/国企项目
