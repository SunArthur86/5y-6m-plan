# 20 - AIoT / 边缘智能 · 5年执行计划

## 总目标
5年在AIoT/边缘AI领域实现年营收600万（硬件+软件+解决方案）。

## Year 2：解决方案复制（年入50万→150万）

### 核心策略
1. **标准化方案**：把定制项目变成可复制方案
2. **渠道合作**：与系统集成商/代理商合作
3. **硬件优化**：自研或定制边缘AI硬件
4. **SaaS平台**：设备管理+模型更新+数据看板
