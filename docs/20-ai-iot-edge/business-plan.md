# 路径 20：AI + 物联网 / 边缘计算 — 商业计划书

## 一句话定位

在边缘设备上部署AI模型（TinyML/边缘推理），为制造业/智能家居/车联网提供AIoT解决方案。

## 市场分析

| 指标 | 数据 |
|------|------|
| 全球AIoT市场 | $500亿(2025) |
| 边缘AI芯片市场 | $200亿 |
| 中国工业AI市场 | ¥3000亿 |
| 年增长率 | 25% |

## 产品方向

### 方向A：工业质检AI（最成熟）
- 产线上的AI视觉检测替代人工
- 边缘部署(延迟<10ms)
- 单条产线收费：¥50-200万
- 每年维护费：¥10-30万
- 客户：制造业/半导体/食品

### 方向B：边缘AI推理平台
- 让AI模型在手机/IoT设备/车机上运行
- 模型压缩+量化+推理引擎
- 类似：TensorRT/ONNX Runtime的商业版
- 定价：¥5-50万/项目授权

### 方向C：智能家居AI中枢
- 本地AI语音助手(不上传云端)
- 隐私优先
- ToC产品+ToB解决方案
