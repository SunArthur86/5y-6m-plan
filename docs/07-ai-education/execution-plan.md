# 07 - AI 教育/培训 · 5年执行计划

## 总目标
5年建立AI教育品牌，年营收600万（课程+训练营+企业培训+会员）。

---

## Year 1：种子期（0→年入30万）

### Q1（月1-3）：内容冷启动

**月1：定位+首批内容**
| 周 | 任务 | 优先级 |
|----|------|--------|
| W1 | 确定细分定位：「给工程师的AI落地课」/「AI应用开发实战」 | 🔴 |
| W1 | 竞品分析：市面top10 AI课程，分析优缺点 | 🔴 |
| W2 | 设计课程大纲（20课时×30分钟） | 🔴 |
| W2 | 录制前3节免费课作为引流 | 🔴 |
| W3 | 在B站/YouTube/小红书发布免费内容 | 🟡 |
| W3 | 建微信群/知识星球（初期免费） | 🔴 |
| W4 | 收集第一批100个学员反馈 | 🔴 |

**月2：付费课程上线**
- W1：完善到10节课，上线付费版（定价499-999）
- W2：设计营销漏斗：免费课→社群→付费课
- W3：首周促销+限时优惠
- W4：收集付费学员反馈，快速优化

**月3：内容矩阵**
- 每周发布2条短视频（B站/抖音/小红书）
- 每月1篇深度长文（公众号/知乎）
- 目标：全网粉丝5000+

### Q2（月4-6）：训练营模式
- 月4：设计第一期训练营（21天AI实战，定价2999）
  - 每天视频课+作业+答疑
  - 招生目标：30人
- 月5：执行训练营+收集testimonial
- 月6：第二期训练营（基于反馈优化）

### Q3（月7-9）：企业培训
- 月7：设计企业培训方案（2天线下/线上）
  - AI战略工作坊（3万/场）
  - AI技术实战营（5万/场）
- 月8：联系3-5个企业HR/培训负责人
- 月9：完成首场企业培训

### Q4（月10-12）：年度目标
- 课程收入 15万+训练营 10万+企业培训 5万 = 30万
- 全网粉丝2万+
- 付费学员300+

---

## Year 2：增长期（年入30万→150万）

### 核心策略
1. **课程矩阵**：3-5门不同课程覆盖不同人群
2. **训练营升级**：从21天→3个月系统班（定价9999）
3. **企业培训常态化**：每月1-2场
4. **会员制**：年费会员（999/年），持续更新内容

### 月度节奏
```
月1-3：新课上线+招生体系化
月4-6：训练营规模化（每期50-100人）
月7-9：企业培训签约（框架协议）
月10-12：会员制上线+年度大促
```

---

## Year 3：品牌化（年入150万→300万）

### 核心策略
1. **讲师团队**：培养3-5个合作讲师
2. **内容授权**：与平台合作（极客时间/掘金）
3. **出版**：写1本AI技术书籍
4. **大会**：主办行业大会/峰会

---

## Year 4-5：收获期

### Year 4（年入300万→450万）
- 品牌+平台：学员自发传播
- 多元收入：课程+训练营+企业培训+会员+书籍+广告

### Year 5（年入450万→600万）
- 教育平台化：不只是个人IP
- 可选融资/合作扩大规模
- 年营收600万
