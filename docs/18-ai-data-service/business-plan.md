## 产品架构

```
┌─────────────────────────────────┐
│   数据需求方(大模型公司)          │
│   → 发布数据需求+验收标准        │
├─────────────────────────────────┤
│   数据中台(我们)                 │
│   → AI预标注+人工精标+质量审核   │
├─────────────────────────────────┤
│   标注员(众包/全职)              │
│   → 按件计酬                    │
└─────────────────────────────────┘
```

**关键壁垒**：AI预标注将效率提升5-10倍，人力成本降60%
