# 路径 16：技术人才平台 / 高端猎头 — 商业计划书

## 一句话定位

用AI匹配+技术人脉做高端技术人才猎头/平台，单笔佣金¥10-50万。

## 市场分析

| 指标 | 数据 |
|------|------|
| 中国猎头市场规模 | ¥2000亿/年 |
| 技术岗猎头佣金 | 候选人年薪的20-30% |
| 高端技术岗年薪 | ¥100-500万 |
| 单笔佣金 | ¥20-150万 |

## 商业模式

### 模式A：高端技术猎头（轻资产）
- 只做年薪¥100万+的技术岗
- AI辅助简历筛选+人岗匹配
- 单笔佣金：¥20-50万
- 目标：每月成交2-3单

### 模式B：AI人才评估SaaS
- 技术能力AI评估（代码测试+系统设计面试）
- 企业订阅使用
- 定价：¥5-20万/年

### 模式C：技术社区+招聘
- 先做技术社区(免费内容引流)
- 再做招聘功能(企业付费)
- 类似：牛客网/LeetCode的中国版
