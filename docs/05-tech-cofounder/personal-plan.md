# 路径 05：技术合伙人 — 个人计划书

## 5年目标

### Y1：找到对的公司+加入
- 每月面试2-3家创业公司
- 评估创始人+市场+团队
- 签约CTO/技术合伙人
- **里程碑**：加入A轮AI公司，拿10%股权

### Y2-Y3：全力以赴
- 搭建技术团队(10-30人)
- 交付核心产品
- 帮公司完成B轮融资
- **里程碑**：估值从¥1亿→¥10亿

### Y4-Y5：退出或继续
- 推动公司走向IPO或被收购
- 或转型为董事会成员
- **里程碑**：个人分得¥3000万+

## 选公司的Checklist

- [ ] 创始人是否靠谱(背景/能力/人品)
- [ ] 市场是否够大(TAM>¥100亿)
- [ ] 是否已有PMF(付费用户>50)
- [ ] 团队是否互补(商业+技术+运营)
- [ ] 股权结构是否合理
- [ ] 投资人是否知名(信号作用)
- [ ] 是否有明确的退出路径

## 关键谈判要点

1. 股权比例(目标10%+)
2. Vesting schedule(4年+1年cliff)
3. 离职时的股权处理
4. 薪资(市场价70-80%)
5. 决策权(技术方向一票否决权)
6. 竞业限制范围

## B计划
- 加入6个月发现不对→及时止损(沉没成本<1年薪资)
- 公司倒闭→技术经验宝贵,回大厂P8+
