# 路径 03：独立开发者 / 海外 SaaS — 商业计划书

## 一句话定位

面向全球市场的垂直 AI 工具 SaaS，追求 MRR $50K+（约¥35万/月，年¥420万）。

## 为什么选海外市场

| 维度 | 国内 | 海外 |
|------|------|------|
| 支付意愿 | ¥10-100/月 | $10-200/月 |
| 竞争 | 激烈+大厂碾压 | 垂直领域机会多 |
| 支付渠道 | 微信/支付宝 | Stripe/Paddle(全球) |
| 语言 | 中文 | 英文(全球70亿) |
| 版权保护 | 弱 | 强 |

## 产品方向（3选1）

### 方向A：AI Code Review Bot
- GitHub App，自动Review PR
- 定价：$29/月(个人) / $199/月(团队)
- 竞品：CodeRabbit($12M ARR), Codacy
- 差异化：深度理解业务逻辑的Review

### 方向B：AI文档自动化
- 录屏→自动生成文档+截图+流程图
- 定价：$19/月(基础) / $99/月(团队)
- 竞品：Tango, Scribe
- 差异化：支持中文+多语言，成本更低

### 方向C：AI客户洞察平台
- 分析客户评价/反馈→自动生成洞察报告
- 定价：$49/月 / $299/月(企业)
- 竞品：较少垂直方案
- 差异化：行业专属分析模型

## 商业模式

| 层级 | 价格 | 目标 |
|------|------|------|
| Free | $0 | 1000免费用户→转化5% |
| Pro | $29/月 | 核心50付费用户→MRR $1450 |
| Team | $99/月 | 20团队→MRR $1980 |
| Business | $299/月 | 10企业→MRR $2990 |

**盈亏平衡：MRR $5000（约60付费用户@Avg $83）**
