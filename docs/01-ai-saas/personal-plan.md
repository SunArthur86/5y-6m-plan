# 路径 01：AI SaaS 创业 — 个人计划书

## 5 年分阶段目标

### Y1（2026-2027）：验证期 — 从想法到PMF
- Q1-Q2：完成MVP，上线核心功能
- Q3：获取前100个付费用户，验证PMF
- Q4：完成种子轮融资（¥500万）
- **里程碑**：ARR ¥200万，团队8人

### Y2（2027-2028）：增长期 — 从PMF到规模化
- Q1-Q2：完成Pre-A轮，团队扩展到25人
- Q3：ARR突破¥1000万
- Q4：建立代理商网络（10+城市）
- **里程碑**：ARR ¥2000万

### Y3（2028-2029）：扩张期 — 行业领导者
- 完成A轮（¥1亿）
- 团队扩展到60人
- 推出3个行业垂直方案
- **里程碑**：ARR ¥1亿

### Y4（2029-2030）：成熟期 — 盈利与护城河
- 实现单月盈亏平衡
- 开放平台生态
- **里程碑**：ARR ¥5亿

### Y5（2030-2031）：退出或继续
- B轮融资或战略收购
- 个人股权价值 ¥5000万-2亿
- 或继续独立运营
- **里程碑**：个人净资产达600万年收入

## B计划（退路）

| 失败阶段 | 退出策略 |
|---------|---------|
| MVP失败(无人用) | 回大厂，走路径02高管路线 |
| PMF失败(有人用但不付费) | 转开源项目，走路径06 |
| 增长停滞(ARR<500万) | 出售给大厂/被收购 |
| 融资失败 | 转自负盈亏模式，缩小团队 |
| Y3 ARR < 5000万 | 转型AI咨询(路径04) |
